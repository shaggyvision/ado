Esta versión conserva IA, Google Sheets, WhatsApp guiado y panel administrativo.

- Compatibilidad con áreas seguras de iPhone/Android.
- Chat desplazable aunque aparezca el teclado.
- Botones y formularios táctiles.
- Menú lateral adaptable.
- Visor de PDF/video a pantalla completa.
- Panel administrativo responsive.
